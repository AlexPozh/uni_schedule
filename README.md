# Из чего состоит проект?
### Парсеры
В проекте используется несколько парсеров для извлечения данных из учебных документов.
#### Учебный план
Парсер, находящийся в модуле `parsers/curriculum_parser.py`, парсит дисциплины, учебную нагрузку, курсы и семестры, на которых читают дисциплины, а также формы контроля. Все эти данные хранятся в **учебном плане** каждого направления подготовки.
